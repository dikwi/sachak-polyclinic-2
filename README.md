# Sachak Polyclinic 2 Website

Official website for Sachak Polyclinic 2 (មន្ទីរពហុព្យាបាល សច្ចៈ២), a trusted healthcare provider in Phnom Penh, Cambodia.

## Features

- **Bilingual Support**: Seamless switching between English and Khmer languages
- **Responsive Design**: Mobile-first design that works on all devices
- **24/7 Healthcare Services**: Specialized in Hepato-Gastroenterology
- **Expert Medical Team**: Featuring Dr. LEAN SOPHEAK, Dr. HOR KOSAL, and Dr. KONG Chheangleng

## Technologies Used

- HTML5
- Tailwind CSS (via CDN)
- Vanilla JavaScript
- Responsive images and modern CSS

## Quick Start

Simply open `index.html` in a web browser to view the website.

For local development with live reload:
```bash
# Using Python 3
python3 -m http.server 8000

# Or using Node.js
npx http-server
```

Then visit `http://localhost:8000`
